## 🚀 Getting Started

### Prerequisites

- List software dependencies or system requirements here:
  - VSCode
  - WSL (Linux) / MSVC (Window)
  - Cmake
  - Gtest
  - AI Copilot
  - gdb

### Source

- CmakeLists.txt: config Cmake
- build/: build system of project
- *.hpp, *.h: header files
- *.cpp: driver source / test source

### Usage

- cd to specific workspace
- cmake -B build
- cmake --build build
- ./build/executableFile
- Debug by gdb in VSCode (F5). For example: Gtest_Analysis_Practices/
